1.11.2. Creating a new segment and assigning ports

To illustrate this process we will create a segment (segment 0) and assign 2 ports to it.
To start the segment creation process, select the option n that appears in the submenu on the right side of the screen.
Once you have reviewed the ports that are available, select the one you want to assign to the segment as it appears in the lower margin of the previous screen.
Enter the segment number on the line Insert segment number (0-15) [0]:
Th number in brackets [] it is the one that defines the system by default. To select this number press Enter.
The port previously assigned to the segment will disappear from the list of available ports.

Note

The system only allows the creation of a maximum of 16 segments (0-15).
