1.1. What is an IDS?

IDS is an Intrusion Detection System to protect one or more organization networks.
It is a specific software that runs in the user space and analyze network traffic (using a mirror port or SPAN) in one system interface or between two interfaces joined as a bridge (software switch).
The traffic is analyzed looking for anomalies, specific signatures or suspicious behavior.
The goal of an IDS is to detect intrusions and create alerts without interfering with the organization's traffic.
