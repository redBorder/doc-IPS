1.9.2. Add a new port to bonding

The system will ask you the following question: Insert second port (y/N)
The default option is displayed in upper case. Remember that if you dont need to add a new port you only need to press Enter.
In case you want to insert a new port press y.
Add new port to bonding
Enter the port number on the line Insert bonding second port [1]:
The number in brackets [] it is the one that defines the system by default. To select this number just press Enter.
