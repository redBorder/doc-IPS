1.5. Burning the ISO Image

The ISO image is prepared to perform the boot from a DVD reader, USB device or as an ISO file for a virtual machine.
NOTE: The process of creating a bootable USB for the IPS installation is the same as for the manager installation. You can find it accessing the support page.
In order to burn the ISO image from a Linux system to a USB device, the following command needs to be used if the USB device is mapped in /dev/sdd: 
[root@machine ~]# dd if=redBorder-3.1.XX-XX-x86_64-6.5-sensor.iso of=/dev/sdd bs=10M

Note

Replace XX-XX in redBorder-3.1.XX-XX-x86_64-6.5-sensor.iso by the patched version of the redborder 3.1 that you have, for example: redBorder-3.1.80-18-x86_64-6.5-sensor.iso.
