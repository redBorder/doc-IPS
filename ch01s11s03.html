1.11.3. Add a second port to the segment

Once you have assigned the first port to the segment, the system will ask you to assign a second port: Assign second port to the segment (Y/n):
The default option is displayed in upper case.
Remember that if you want to assign a new port you only have to press the key Enter.
In case you dont want to assign a new port, type n. Select an available port from the list.
Enter the port number on the line Insert segment second port [3]:
Assign a new port to the segment
Once both ports have been assigned to a segment, the system will notify you that the segment has been successfully created: Segment 0 created successfully.
