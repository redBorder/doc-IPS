1.2. What is an IPS?

IPS is an Intrusion Prevention System that works in the network or networks inside an organization.
It works in the same way as an IDS (in fact, it is usually the same software with an adapted behavior), except that in this case it does interfere with the organization's traffic, in that way when an attack is detected the system can choose between showing alerts or even discard the packets, creating the same effect as a firewall.
The configuration and registration of the IPS/IDS sensors into the Manager is the next step to the installation one. This process consists in two steps:
  • Installation and registration of the IPS/IDS sensors in the Manager
  • Configuration of basic parameters of the IPS/IDS sensors in the redborder platform
