1.9.3. Bonding network configuration

Once you have assigned the second port to the new bonding (bond0), the system will ask you to enter the following data:
  • IP Address
  • Network mask
  • Gateway
  • Routes
If your device uses the DHCP protocol, it will be assigned a default IP address. Press Enter to select the IP, mask and gateway assigned by the system. Otherwise, manually enter these values.
The system also gives you a default route for the created bonding.
To insert a route manually, select the option y on the line Insert a route for this bonding (y/N)?:
Manually add a route to the bonding
Once you have finished this process, you will see a message advising you that the bonding has been successfully created: Bonding 0 created successfully.
