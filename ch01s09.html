1.9. Interface assignment and bonding creation

Select Network Configuration typing 2 and press Enter.
Automatic assignment of segments with bypass
In case the system recognizes interfaces with bypass, these will be automatically assigned to the segments.
Network sensor configuration
To configure the network configuration correctly:
  • Bonding creation: (maximum 4 bonds).
  • Selection and assignment of bonding ports.
  • Management network interfaces selection for bonding.
The system will show a list with very valuable and detailed information, which will help the administrator to decide which ports are available for the creation of the bonding.
Insertar bonding port

Note

To select the default options press: Enter

1.9.1. Creation of a new bonding and port assignment

To illustrate this process we will create a bonding (bond0) and assign 2 ports to it.
Select option n that appears in the submenu on the right side of the screen.
Once you have reviewed the ports that are available, select the one that you want to assign to bonding, as it appears in the lower margin of the previous screen.
Enter the port number on the line Insert bonding first port [0]:
The number in brackets [] it is the one that defines the system by default. To select this number press Enter.
Bonding creation: port assignment
The selected port will disappear from the available ports list.

Important

The system only allows a maximum of 4 bonds: (0-3).
