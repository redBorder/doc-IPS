2.1. Registration of the sensor in the Manager

To finish the installation process of the IPS/IDS sensor, the last step consist in Manager sensors registration.
Until the registration is complete, a message will be displayed indicating that the sensors have not yet been registered in the Manager.
Starting sensor registration to the Manager
In the sensor registration menu we can find the following options:
  1. Hostname [rbsensor]: assign a sensor name.
  2. Set local time: configure local time.
  3. IP adress for rB Master Manager [127.0.0.1]: insert the Manager IP.
  4. Register rB/Manager: register the sensor into the Manager.
  5. Register to a redborder cloud: register the sensor into redborder live platform.

2.1.1. Assign sensor name (Hostname)

Select the option Hostname [rbsensor] from the registration menu by pressing 1.
Next, on the line Insert Hostname [rbsensor]: type the name you want to give to the sensor. From now on, this is the name that will be displayed on the web when you access to the list of IPS sensors.
Assign a sensor name

Note

Remember to enter a consistent name for the sensor. Avoid using the name “Sensor”, as this is the generic name used by the Manager and could cause confusion in the long run.

Warning

The system will ask you to verify that the chosen name is correct, in this case, press Enter
If you wish to modify it, insert the letter n
The system will proceed to change the name of the sensor. To continue, and return to the redborder system configuration menu, press any key as indicated Press a key to continue...

2.1.2. Local time configuration

Select option 2) Set local time.
Insert date and time of your country.
The system indicates two possible formats for inserting the date and time:
  • [HH:mm] or [YYYY/MM/DD]
  • [YYYY/MM/DD HH:mm]
Press Enter to continue with the registration process.
Sensor registration in the Manager: Select local time

2.1.3. Insert the Master Manager IP

Select option 3) IP adress for rB Master Manager [.*].
Insert the IP address or domain name of the computer used by the user. The system will perform a series of checks to verify if it is reachable. This process may take a few seconds.
The system will synchronize the time of the computers and then restart a couple of services in order to finish the time synchronization.
Congratulations! Your machine is ready to register the sensor in the Manager.
Insert Master Manager IP
To continue, and return to the redborder system configuration menu, press any key as indicated Press a key to continue...

2.1.4. Registration of the sensors in the Manager

This is the last step to configure your redborder IPS/IDS sensor.
From this moment on, you will be able to work directly from the IPS management application, in whose dashboard you will graphically view all the traffic events captured by the IPS sensors.
In this application you can create reports, view events or configure the rules of the Manager among other actions.
Completing the sensor registration process in the Manager
The steps to follow to complete the sensor registration process in the Manager are as follows:
  1. Indicate if you want to have a debugging output: the system will ask the following question Would you like to have a debug output? Select default option (No) pressing the key N.
  2. Insert password: indicate the password that has been entered in the basic settings phase for the configuration of the Manager.
  3. Time synchronization (automatic) between the Manager and the sensors: this process may take a few seconds to complete.
Completing the registration of the sensors in the Manager
After completing these steps, the system will notify you with a message that the sensor has been successfully registered.
Sensor registered succesfully with chef server
Sensor registration on the server

Warning

Do not change the sensor name or hostname once it has been registered. If you want to make changes, you will need to undo the entire registration process and start over.
To continue and return to the redborder system configuration menu, press any key as indicated Press a key to continue...
In the sensor registration menu in the redborder Manager you can check that the sensor has been successfully registered:(Registered).
Sensor registration successful
To exit the sensor registration wizard, select option quit pressing q.
