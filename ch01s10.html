1.10. DNS and domain configuration

In the network configuration menu we find the option DNS and domain settings. To start the process, select option2.
To perform the configuration correctly, the user must enter:
  • Domain name
  • Primary DNS address
  • Secondary DNS address (optional)
Once this information has been entered, the system will warn that this configuration has been applied correctly.
DNS and domain configuration
To finish the process, it is not necessary to apply changes, select the option quit pressing q to return to the network settings main menu.
