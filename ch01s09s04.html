1.9.4. Finish the bonding creation process

Before applying changes and finishing the bonding creation process, the system will show you a table that shows the newly created bonding (bond0) together with the ports that have been assigned to it, the assigned IP and Mask and the route.
Apply changes
To save the settings select option a.
You can delete the created bonding(s) at any time by selecting the option delete bonding pressing the key d.
To return to the main network configuration menu, select the option quit pressing q.
