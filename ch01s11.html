1.11. Network segments configuration

Select in the Network Configuration Menu the option Segment settings pressing the number 3.
This screen shows the number of segments that have been created, the ports assigned to each segment and whether or not they have bypass support.
On this section you will find the following submenu:
  • f) force bypass autoassign: to automatic assignment of ports with bypass.
  • n) new segment: select to create a new segment.
  • d) delete segment: to delete a segment.
  • q) quit:exit.
To start creating the segment, select the option n) new segment.
The segment creation process includes the following actions:
  1. Assignment of bypass interfaces to segments (optional).
  2. Create a segment: (max 16 segments).
  3. Selecting and assigning ports to the segment
In order to select a port, a list with very valuable and detailed information will be displayed, which will help the administrator to decide which ports the created segment will use.

1.11.1. Bypass interfaces selection

If the system detects bypass interfaces, they are automatically assigned to segments when you enter the network configuration menu.
There are also two other ways of assigning ports or interfaces with bypass to segments:
  • Force bypass autoassign: select option Force bypass auto assign in the segment configuration menu, entering the letter f.
  • Manual assignment: selecting the ports with bypass from the list of available ports that will be displayed in the process of creating the segments.
