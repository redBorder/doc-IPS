1.11.4. Finish the process of creating the segments

Before applying changes and finishing the segment creation process, you will see a screen with a list that includes the following elements:
  • Segments
  • Ports assigned to the segment
  • Whether or not it has bypass support
Summary of assigned segments and ports
You can delete in any moment the created segments by selection delete segment, pressing d.
To return to the main network configuration menu, select the option quit by pressing q.
