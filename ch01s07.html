1.7. Basic Configuration

Note

To make this installation and configuration process it is necessary to login with the user with highest permissions: root.
Start the configuration manager by executing: 
[root@rbmanager ~]# rb_sysconf
Figure 5: Basic Configuration
This menu consists of 5 options:
  1. System Configuration: from this option you can define basic settings such as hostname, date, time, etc.
  2. Network Configuration: this is the option where you can configure the network settings.
  3. Passwords: access this option if you want to change your passwords.
  4. Update rules from Manager: select this option to force the download of the Manager rules (optional).
  5. quit: exit.
