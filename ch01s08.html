1.8. Network configuration

IDS/IPS sensors need to comunicate with the Manager, to make this possible it is necessary to make the network configuration. Select option 2) Network Configuration from the redborder configuration menu.
Until this setting is not made properly, it will not be possible to register the sensor in the Manager.
These are the steps you need to follow to configure the network of your sensor:
  1. Management Network Configuration:
    a) Bonding creation and management interface assignment.
    b) Segments configuration.
  2. DNS and domain configuration

Note

The order to configure the management network can be different. The order indicated in this installation manual is the one considered most optimal.

Warning

In case that the system is installed on a real computer that has more than one network card, the user must choose which will be the management bonding.
